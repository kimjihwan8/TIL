# 6. 차이점

| **구분**     | **Workspaces**             | **Lerna**               | **Turborepo**       | **Nx**                  |
| ---------- | -------------------------- | ----------------------- | ------------------- | ----------------------- |
| **역할**     | **기반 기술**                  | **배포/버전 관리 특화**         | **빌드 속도 최적화 특화**    | **종합 관리 플랫폼**           |
| **제공 주체**  | npm, yarn, pnpm 자체 기능      | 별도 CLI 도구               | 별도 CLI 도구           | 별도 CLI 도구               |
| **주요 목적**  | 폴더 간 연결 (심볼릭 링크),의존성 통합 설치 | 여러 패키지 동시 배포,버전 관리 자동화  | 초고속 병렬 빌드,지능형 캐싱    | 코드 생성 표준화,영향도 분석,빌드 최적화 |
| **속도/성능**  | 기본 설치 기능                   | 과거 느림 → 현재 빠름 (Nx엔진 탑재) | **매우 빠름 (Rust 기반)** | 매우 빠름 (캐싱, 분산 빌드)       |
| **학습 난이도** | 쉬움                         | 보통                      | 쉬움 (단순 설정)          | 높음 (기능 방대)              |
